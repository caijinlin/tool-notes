    编辑

    删除

    • ctrl + d/ctrl + h 删除当前光标、当前光标之前的字符
    • ctrl + w 删除光标之前的单词
    • ctrl + k 删除到文本末尾
    • ctrl + u 清除当前行

    替换

    • ctrl + t 交换光标处文本

    查找

    • ctrl + p 上一条命令
    • ctrl + r 查找历史命令
    • command + ; 自动列出之前输入过的类似命令

    撤销

    • ctrl + - 撤销
