# 移动

### 水平移动

* `hjkl` 上下左右
* `0/^/$` 移动到行首、行中第1个不为空的字符、行尾
* `b/B` 跳转到上一个单词开头
* `w/W` 跳转到下一个单词开头
* `e/E` 跳转到下一个单词结尾

### 垂直移动

* `(/)` 句子之间移动
* `{/}` 段落之间移动
* `gg/G` 移动到文件开头结尾
* `nG` 移动到第几行
* `H/M/L` 屏幕开头、中间、结尾
* `Ctrl+u/Ctrl+f` 向上向下翻页
* `zt/zz/zb` 把当前行设置到顶部、中部、底部

### 历史移动

* `ctr-o` 回到之前定位
* `ctrl-i` 回到最新定位
