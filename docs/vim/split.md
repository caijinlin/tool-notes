# 分割屏幕

### 分屏

* `vim -On file1 file2` 左右分屏打开文件
* `vim -On file1 file2` 上下分屏打开文件
* `Ctrl +W v` 左右分屏当前文件 
* `Ctrl +W s` 左右分屏当前文件 
* `vsp file` 左右分屏打开新文件
* `sp file` 上下分屏打开新文件
* `Ctrl + w/h/j/k` 切换分屏


### 标签

* `vim -p file1 file2` tab标签打开文件
* `:tabnew` 打开新的tab
* `:tabclose` 关闭当前的tab
* `:tabonly` 关闭所有其他的tab 
* `:tabprev` 前一个tab
* `:tabnext` 后一个tab
* `gt` 切换标签
